### Installation / Usage
1. Install linux (supports only linux for now).

2. From repository root, run:

   ```Shell
   mkdir build && cd build
   ```


3. Build the program: 
   ```Shell
   cmake .. && make
   ```
  4. Run the executable:

     ``` 
     ./bin/ecs
     ```
